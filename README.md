# Tmux Config

## Installation

After cloning, create a symlink at `~/.config/tmux/tmux.conf` to the configuration file in this repo

Also install tpm

```shell
git clone https://github.com/tmux-plugins/tpm ~/.tmux/plugins/tpm
```

Then resource configuration file

```shell
tmux source ~/.tmux.conf
```

And you're good to go!
